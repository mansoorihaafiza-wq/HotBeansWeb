Join the team — job specifications

We’re looking for motivated developers with relevant training or degrees. Explore roles below.

👨‍💻 Junior Web Developer

Salary: £28k - £34k

Requirements: HTML, CSS, JavaScript basics, IT diploma or degree (or equivalent bootcamp).

Full spec: Build landing pages, collaborate with designers, write semantic HTML/CSS, participate in code reviews. Qualification: Level 3+ IT / web development course or proven portfolio.

⚛️ React Developer (Trainee)

Salary: £30k - £38k

Requirements: React basics, ES6, Git, and a portfolio or certification (e.g., Meta Frontend).

Full spec: Develop component libraries, manage state, integrate APIs. Qualification: Completion of recognised React course OR 1+ year hands-on.

📱 Junior Backend Developer

Salary: £29k - £36k

Requirements: Node.js/PHP basics, SQL, understanding of REST APIs. Degree or bootcamp graduate.

Full spec: Build server-side logic, database design, API endpoints. Desired: any cloud fundamentals. We provide mentorship.

Minimum required qualification: Level 4 Diploma in Web Development / IT or equivalent industry experience. We welcome self-taught devs with strong portfolios!
